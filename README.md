# Pika Pika
A lite version of SNS where everyone can share their memories. PikaPika üòç

## Features
Users can
- Login / Register / Logout
- Upload a new Post
- Delete their own Post
- View the timeline
- Pull to refresh new data

## Environment
- Xcode 13.4
- Swift 5
- SwiftPackageManager

## 3rd party libraries
- FirebaseAuth
- FirebaseFireStore
- FirebaseStorage
- Kingfisher
- SnapKit

## To test
- Fork or clone the repository
- Build & Run (after SPM finishes fetching)
